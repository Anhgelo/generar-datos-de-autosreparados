# generar-datos-de-autosreparados
El siguiente script en Python genera fake data de un centro de reparación de automóviles, con el objetivo de dar un análisis de datos a una lista masiva
En este codigo es necesario instalar primero 

pip install streamlit faker openpyxl xlsxwriter

tambien los vehiculos
pip install faker_vehicle

al correr el archivo usa 
streamlit run app.py
